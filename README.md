## About
- st is a simple terminal emulator for X which sucks less created by [suckless guys](https://suckless.org/). 

## Requirements
- Xlib header files.

## Installation
```sh
git clone https://github.com/pwhitecat/st ~/.local/share/st
cd ~/.local/share/st
sudo make clean install
```
